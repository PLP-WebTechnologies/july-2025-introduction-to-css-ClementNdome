Introduction

CSS allows us to style HTML elements, control layout, and design visually appealing web pages. A strong foundation in selectors, spacing, and sizing is essential for building consistent user interfaces.

The CSS Box Model

Every element in CSS is treated as a rectangular box, composed of:

  • Content – the text or image inside the element
  • Padding – space between content and border
  • Border – wraps around padding and content
  • Margin – space outside the border
